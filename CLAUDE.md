# Programação Web 1 — Quiz de Estudo

## Stack
- HTML + CSS + JavaScript puro (vanilla), sem dependências externas
- Arquivo único standalone (`index.html`) + GitHub Pages

## Estrutura
- `index.html` — o quiz interativo (banco de 20 questões embutido no próprio JS)
- `prints/` — 20 prints originais das questões da prova (material-fonte)
- `README.md` — descrição + link do site
- `.gitignore` — lixo de sistema e pasta temporária `_crops/`
