# 📚 Programação Web 1 — Quiz de Estudo

Material de estudo interativo para a disciplina de **Programação Web 1**, montado a partir das questões da prova. Um quiz único, em HTML puro, que roda offline (é só abrir o arquivo) ou online pelo GitHub Pages.

🔗 **Estudar agora:** https://kahuasantos1-ui.github.io/programacao-web-1-estudo/

---

## 🎯 O que tem

São **20 questões** com correção na hora e explicação de cada alternativa, divididas em 4 blocos:

| Bloco | Questões | Conteúdo |
|---|---|---|
| 🔵 **CSS** | 5 | formas de definir CSS, bloco de declaração, tipos de seletores, `float`, seletor universal |
| 🟡 **JavaScript** | 5 | função × evento, eventos `On*`, métodos `Alert/Confirm/Prompt`, sintaxe `<script>`, variáveis |
| 🔵 **jQuery** | 5 | características da lib, selectors (`#` `.` `*`), instalação, função `$`, `.hide()` |
| 🟢 **Frameworks** | 5 | Bootstrap × Angular × Vue.js, Virtual DOM, gerenciamento de estado, escalabilidade |

---

## ⚙️ Como usar

- **Clica na alternativa** → mostra na hora se acertou (verde) ou errou (vermelho) **e o porquê** de cada opção.
- **Filtro por tópico** no topo — estuda só CSS, só JS, etc.
- **🔀 Embaralhar** — muda a ordem pra não decorar a sequência.
- **📖 Modo revisão** — mostra todas as respostas certas direto, ideal pra revisão rápida antes da prova.
- **Pontuação e barra de progresso** acompanhando acertos e erros.

> Não precisa de internet nem instalar nada. É um único arquivo `index.html` standalone.
